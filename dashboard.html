システム概要

  • 送り先・車両マスタを事前登録
  • 予定表で送迎計画を作成
  • 当日運用で自動配車・手動調整
  • 実際の送りで実績管理
  • 月間走行距離・出勤日数を集計
  • LINE連携でドライバーへ通知
  • GoogleMapでルート確認可能

システム仕様

  • 住所入力 → Enterで座標取得
  • 同一住所はジオコードキャッシュ
  • 距離は起点から自動計算
  • シミュレーションで便別試算可能
  • 完了送り先は次回配車から除外
  • 配車後の車両変更可能
  • CSV出力対応

現在のプラン

チーム単位のプラン情報です。現在は起点・車両・送り先の上限チェックと、CSV機能のPaid制御が有効です。

課金を開始すると Stripe の決済画面へ移動します。反映後はプランを再読込してください。

ユーザー管理

オーナー専用です。一度ログインしたユーザーが一覧に表示されます。ここで表示名・権限・有効状態を管理します。

ユーザー招待

招待制で運用します。オーナーは admin / user を招待でき、管理者は user のみ招待できます。第1段階では招待レコードを作成し、状態を管理します。

pending / accepted / expired / revoked を管理します。

起点管理

起点は5件まで保存できます。使いたい起点を選ぶと、以後の距離計算・地図・配車の基準に反映されます。

住所から GoogleMap を開いて位置を確認し、座標または Google Maps のURLを貼り付けて保存できます。送り先管理では距離と時間を固定保存せず、現在の起点から都度再計算します。
